## JSP MVC2 공부하면서 만들어본 가상의 저축 웹

- JSP MVC2 모델을 실습하기 위해 진행한 프로젝트로 무료 템플릿을 가져와 프론트딴 코드는 난잡하다...((css.js)불필요한 부분,파일도 많고 난잡하게 사용함),,
